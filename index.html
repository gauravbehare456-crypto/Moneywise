Google Drive Setup
Google Drive backup साठी एक OAuth 2.0 Client ID लागतो.

Steps:
1. console.cloud.google.com उघडा
2. नवीन Project → Google Drive API enable करा
3. Credentials → OAuth 2.0 Client ID → Web application
4. Authorized origins: तुमचा URL किंवा http://localhost
5. Client ID copy करा खाली paste करा
